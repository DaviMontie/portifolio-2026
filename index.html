Seja bem-vindo(a) ao meu portfólio acadêmico. Este site foi desenvolvido com o objetivo de apresentar minha trajetória na disciplina, reunindo informações sobre mim, minhas habilidades atuais em desenvolvimento web e os projetos realizados ao longo das aulas.

Esta página faz parte de uma atividade acadêmica voltada para a prática da construção de páginas web simples, utilizando conceitos fundamentais de estrutura, navegação e organização de conteúdo. Além disso, o projeto também tem como finalidade incentivar o uso do GitHub e a publicação de páginas por meio do GitHub Pages.

Ao navegar pelo menu, você poderá conhecer um pouco mais sobre meu perfil, minhas habilidades atuais e acessar os projetos desenvolvidos durante a disciplina.
